# Python Django CRUD day 2
## DB연결
1. django project 생성
```shell
sudo pip3 install django
```
2. 장고 실행 확인
```shell
python3 manage.py runserver
```
3. 자동생성된 프로젝트 디렉토리 안 settings.py 파일에 디비 정보 입력

4. db클라이언트 설치
```shell
pip3 install mysqlclient
```

console에서 정상적으로 프로젝트가 작동하는지 확인.
```shell
python3 manage.py runserver
```
